GillyCricket

It Plays for you rapidly

Welcome to GillyCricket Page.

This is an Free Software licensed under GNU GPL which fetches Live Cricket scores from Internet faster even in slower internet connections. This Software is powered by Python (a dynamic scripting language) + Tkinter(a GUI development tool).

Available for:

1. GNU/Linux (Ubuntu, Fedora, LinuxMint).

2. Windows 7 (64-bit).

Screenshots:

How to Install and Use:

Download either tar.gz file or .zip file from the repository

1. GNU/Linux Users:-

Extract the files, Open the terminal and move to the extracted directory. Make sure you have installed Python Version 2.7 or below and not 3.0 or later.

$ cd your_extracted_dir/Gilly

If you are running Fedora or its derivative run this:

$ sudo sh fedora_install.sh

If you are running Ubuntu or Debian based derivative run this: 

$ sudo sh ubuntu_install.sh

When everything is done in terminal you can close it and double click the Gilly file and click on Run.

  • You can delete the Downloaded and Extracted folder or directory and you can yet run the program from terminal:
$ Gilly &
  • To get scores only via Notifications:
$ Gillylive time_in_minutes
$ Gillylive 2

Pressing Ctrl + z or closing the terminal should stop this.

The above code will refresh scores every 2 minutes interval time via notifications alone, so that you can carry on with your work while notifications will show you updates in those interval.

2. Windows 7 (64 bit users):

  1. Extract the files
  2. Find Gilly.zip
  3. Extract Gilly.zip
  4. Run Gilly.exe

Uninstalling under GNU/Linux:

Open your terminal and run the below command

sudo removeGilly
